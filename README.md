# Drug Molecule Analyzer

## Description
A beginner-level project that analyzes basic properties of drug molecules using computational methods.

## Features
- Input molecule data in SMILES format
- Analyze properties (e.g., molecular weight, structure)
- Output results and observe displayed structure

## Rationale
I built this project to explore the intersection of chemistry and programming, and to better understand how computational tools can be used in drug analysis.

## Technologies Used
- Python (VS Code)
- RDKit

## How to Run
Run `main.py` → input data (SMILES format) → view output and optional menu features

## Notes
You may download molecules in SMILES format from: https://go.drugbank.com/
